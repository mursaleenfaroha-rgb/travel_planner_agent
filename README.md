## Agentic AI-Based Travel Planner

This project builds an intelligent travel planning assistant using LangChain.

### Features
- Flight selection from dataset
- Hotel recommendation
- Places discovery
- Budget estimation
- Agentic reasoning

### Tech Stack
- Python
- LangChain
- OpenAI
- Open-Meteo API


###project overview:
This project implements an Agentic AI-based travel planning assistant that helps users plan trips by automatically selecting flights, recommending hotels, suggesting tourist places, and estimating the total travel budget. The system uses modular tools and structured datasets to simulate intelligent decision-making similar to a real travel planner.



###How to Run (CLI):

pip install -r requirements.txt
python main.py


Then enter:

-Source city
-Destination city
-Number of days



###Use Cases:

Automated trip planning
Budget estimation for travel
